# Мастерская Вадима Дубинина — Лендинг

Одностраничный адаптивный лендинг для мебельного производства корпусной мебели на заказ.

## Запуск

Откройте `index.html` в браузере или запустите локальный сервер:

```bash
npx serve .
```

## Структура

- `index.html` — разметка
- `css/style.css` — стили, анимации, адаптив
- `js/script.js` — скролл-анимации, мобильное меню, форма
- `assets/` — hero-bg.png, logo.png
